## General Info

Project RISCII (pronounced "risky"- not RISC-II) is a personal project focused on designing the breadth an embedded system from scratch. The primary 'levels' include:

- **Level 5:** Traditional OS Concepts (virtual memory, file system, processes)
- **Level 4:** Firmware (drivers, libraries, RTOS, bootloader)
- **Level 3:** Compiler + Tools (programming language, debug tools)
- **Level 2:** Softcore Microprocessor (processor, peripherals, debug ports)
- **Level 1:** Circuit Board (part selection, wire routing, PCB design)


This project is ultimately for personal enjoyment. ***However***, if there is educational or commerical value, it is free for use (albeit at the user's risk).
